# Article Layout

This page uses `layout: article` - notice the centered content area, readable width, and typography optimized for prose. This is the **default layout** when no layout is specified.

## When to Use

Use the article layout for:
- About pages
- Team information pages
- General informational content
- Documentation pages (like this one)

## Frontmatter

````yaml
---
title: Your Page Title
layout: article  # Optional - article is default
section: about   # Which nav item to highlight
description: SEO description for search engines
keywords: relevant, keywords, for, seo
---
````
